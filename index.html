Este sitio contiene las prácticas de la asignatura de Visión para Robótica del Máster Oficial en Visión Artificial.

Las prácticas de la asignatura se han desarrollado con las siguientes características:

  1. Unibotics: Entorno para el desarrollo de las prácticas.
  2. JdeRobot: Entorno de comunicación entre el usuario y el robot.
  3. Python: Lenguaje para la programación de los robots.
  4. Discourse: Foro para poner en común dudas, problemas y resultados.
  5. GitHub:Como sitio web donde exponer los avances de las prácticas.
